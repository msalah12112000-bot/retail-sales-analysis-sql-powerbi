# E-Commerce Sales Analysis (SQL & Power BI)

## Overview
End-to-end data analysis project using SQL Server for cleaning & transforming raw sales data, and Power BI for interactive dashboards.

## Dataset
- Source: Online Retail CSV
- Includes invoices, products, customers, quantities, prices, and countries

## Project Structure
- `data/` – raw CSV  
- `sql/` – SQL scripts: create DB, staging tables, data cleaning, deduplication, analytical views  
- `powerbi/` – Power BI dashboard

## KPIs
- Total Revenue  
- Monthly Sales Trend  
- Top-Selling Products  
- Sales by Country

## Tools
SQL Server | SSMS | Power BI
